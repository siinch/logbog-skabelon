Recommended HTML Exercises

These exercises will introduce you to the most important aspects and features of HTML. After completing these exercises, you will be able to make a simple website with text, images and videos from scratch. These exercises are for complete beginners and the only thing you need to get started is a text editor. Choose whichever text editor suits you. Visual Studio Code is a very popular choice. After picking a text editor, use the text editor to open and edit the html.html-file, which is the file you are viewing right now. It can be found within the client/exercises sub folder of the project. Save your changes to the file (press control + s or command + s) and open/reload it in a browser window to see the changes.
